# AI Resume & Cover Letter Generator

## 📌 Overview

AI Resume & Cover Letter Generator is a Streamlit-based application developed as part of my Summer Internship at LinuxWorld Informatics Pvt. Ltd.

The application helps users generate professional resumes and cover letters based on their skills, experience, and target job role.

---

## 🚀 Features

* Resume Generation
* Cover Letter Generation
* ATS Score Display
* Resume Download Option
* User-Friendly Interface
* Professional Resume Formatting

---

## 🛠️ Technologies Used

* Python
* Streamlit
* Prompt Engineering Concepts
* Generative AI Concepts

## ⚙️ How to Run

1. Install Python
2. Install Streamlit

```bash
pip install streamlit
```

3. Run the application

```bash
streamlit run app.py
```

4. Open the local URL shown in the terminal.
